# 7TVEmoteDownloader
A little script that downloads every emote of a 7TV User using webscraping (chromedriver).

## Dependencies
You will have to install [google-chrome](https://www.google.com/chrome/) or [chromium](chromium.woolyss.com/download/) AND [chromedriver](https://sites.google.com/chromium.org/driver/downloads) with the **same** version.

## Build dependencies:
``rust cargo``
